### Meteor Starter

A Meteor boilerplate

### Usage

    $ meteor

### Insert data in the Mongo database

    $ meteor mongo -> db.tasks.insert({ text: 'Hello world!', createdAt: new Date() });

### Reset Meteor database

    $ meteor reset
